dirty_cat: machine learning on dirty categories¶

dirty_cat is a small Python module to perform machine-learning on non-curated categories. In particular, it provides encoders that are robust to morphological variants, such as typos, in the category strings.

The SimilarityEncoder can be used as a drop-in replacement for the scikit-learn class sklearn.preprocessing.OneHotEncoder.

API documentation¶

Base classes¶

SimilarityEncoder([similarity, ngram_range, …]) Encode string categorical features as a numeric array.
TargetEncoder([categories, clf_type, dtype, …]) Encode categorical features as a numeric array given a target vector.
