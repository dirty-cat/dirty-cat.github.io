dirty_cat.datasets.fetch_employee_salaries¶

dirty_cat.datasets.fetch_employee_salaries()[source]¶

fetches the employee_salaries dataset

The employee_salaries dataset contains information about annual salaries (year 2016) for more than 9,000 employees of the Montgomery County (Maryland, US).

Returns

a dictionary containing:

  • a short description of the dataset (under the DESCR key)

  • the tabular data (under the data key)

  • the target (under the target key)

Return type

dict

References

https://catalog.data.gov/dataset/employee-salaries-2016
